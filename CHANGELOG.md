# Changelog

## 3.0.0 - 2026-07-23

- Established CodeSync as a public MIT-licensed open-source project with preserved provenance.
- Replaced the inherited streak presentation with a distinct seven-day activity dashboard.
- Fixed local-calendar streak tracking and date-boundary behavior.
- Added Codeforces syncing and optional separate repository support.
- Migrated legacy storage keys to CodeSync-owned names without losing settings.
- Reduced LeetCode GraphQL collection to fields required for uploads.
- Expanded upload credential detection and retained trusted-context-only token storage.
- Added pinned CI actions, dependency review, CodeQL, security reporting, and community templates.
- Bundled license and third-party notices with production builds.
