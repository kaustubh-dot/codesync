# Contributing to CodeSync

## Reporting Bugs

Open an issue in [CodeSync](https://github.com/kaustubh-dot/codesync/issues) with:

- Steps to reproduce the issue
- Expected behavior
- Actual behavior
- Screenshots (if applicable)

## Contributing Code

If you want to contribute code to CodeSync, follow these steps:

1. Create a focused branch.
2. Run `npm ci` and `npm run verify`.
3. Add focused tests for changed behavior.
4. Open a pull request describing security and permission changes explicitly.

## Code of Conduct

CodeSync follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/0/code_of_conduct/).
